# AI LP Generator

AIを活用してランディングページ（LP）を自動生成・編集できるWebアプリです。

## デモ

デプロイURL：
https://あなたのURL.vercel.app

## スクリーンショット

（あとで画像を追加）

## 主な機能

- 🤖 OpenAIによるLP自動生成
- 🖼️ Stability AIによるHero画像生成
- ✨ AIによるLP文章の改善
- ✏️ LP編集機能
- 📷 Hero画像変更
- ↕️ ドラッグ&ドロップによる並び替え
- 🎨 テンプレート切り替え
- 📱 レスポンシブ対応
- 🔗 OGP対応
- 🗑️ LP削除機能

## 使用技術

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- Prisma
- PostgreSQL (Supabase)
- Clerk
- OpenAI API
- Stability AI API
- dnd-kit
- shadcn/ui
